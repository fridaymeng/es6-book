### New string features

```js
> 'hello'.startsWith('hell')
true
> 'hello'.endsWith('ello')
true
> 'hello'.includes('ell')
true
> 'doo '.repeat(3)
'doo doo doo '
```

```js
for (const ch of 'abc') {

}
// Output:
// a

// c
```
